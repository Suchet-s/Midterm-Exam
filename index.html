เรียนรู้ Arrays ใน Python

ศึกษาการใช้งาน Arrays (List) ใน Python พร้อมตัวอย่างและแบบฝึกหัดที่เข้าใจง่าย

การใช้งาน Arrays ใน Python

การสร้าง Array (List) ใน Python

ใน Python, Arrays หรือ Lists เป็นโครงสร้างข้อมูลที่สามารถเก็บข้อมูลหลายประเภท เช่น ตัวเลข (int), ข้อความ (str), หรือแม้กระทั่ง List อื่นๆ

my_array = [1, 2, 3, 4, 5]

การเข้าถึงสมาชิกใน Array

การเข้าถึงสมาชิกใน Array หรือ List สามารถทำได้โดยการใช้ดัชนี (index) ซึ่งเริ่มต้นจาก 0

print(my_array[0])  # Output: 1

การเพิ่มสมาชิกใน Array

การเพิ่มสมาชิกใน Array สามารถทำได้หลายวิธี เช่น การใช้ append() หรือ insert()

my_array.append(6)
my_array.insert(2, 99)

การลบสมาชิกจาก Array

การลบสมาชิกจาก Array สามารถทำได้โดยใช้ remove(), pop(), หรือ del

my_array.remove(99)
removed_value = my_array.pop(1)

การทำงานกับ 2D Arrays (List of Lists)

เราสามารถใช้ List of Lists (หรือ 2D Arrays) ในการจัดเก็บข้อมูลที่มีหลายมิติ เช่น เมทริกซ์

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

การใช้ Loops กับ Arrays

คุณสามารถใช้ Loops เพื่อทำงานกับ Arrays หรือ Lists ได้ เช่น การทำซ้ำเพื่อเข้าถึงสมาชิกทั้งหมดใน List

for item in my_array:
    print(item)

การใช้ List Comprehension

List Comprehension เป็นวิธีที่สะดวกในการสร้าง List ใหม่จาก List เดิม โดยการใช้เงื่อนไขหรือฟังก์ชันต่างๆ

even_numbers = [x for x in range(10) if x % 2 == 0]
print(even_numbers)  # Output: [0, 2, 4, 6, 8]
