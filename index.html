What is an Array?

An Array in Python is a collection of elements, typically of the same type, stored in contiguous memory locations. Arrays are particularly useful when you need to store multiple values and perform operations on them. Python does not have built-in support for arrays, but we can use the array module or lists to work with arrays.

Note: In Python, lists are often used as arrays. However, for performance and memory efficiency, the array module can be used if the type of elements is fixed.

Arrays in Python - Examples

1. Creating Arrays using the array module

To use arrays, first import the array module.


import array

# Creating an array of integers
arr = array.array('i', [1, 2, 3, 4, 5])
print(arr)

2. Accessing Elements in an Array

Elements in an array can be accessed via their index, just like lists.


print(arr[0])  # Output: 1
print(arr[3])  # Output: 4

3. Modifying Elements in an Array

You can modify an element of the array by assigning a new value to it.


arr[2] = 10
print(arr)  # Output: array('i', [1, 2, 10, 4, 5])
